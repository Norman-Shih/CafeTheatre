你怎麼喝咖啡？

不同的沖煮方式，適合不同特性的豆子。

☕
義式機 / 膠囊機
用高壓萃取，適合中深焙。奶咖或直飲都行。
✓
🫗
手沖 / 濾杯
慢萃，風味細節更清楚。從淺到深烘都合適。
✓
🫖
摩卡壺 / 法壓壺
口感厚實，適合中深焙。喜歡濃一點的人。
✓
🧊
冷泡 / 冰咖啡
低溫長時間萃取，偏甜、低酸，很好喝。
